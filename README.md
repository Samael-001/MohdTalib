## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express


## Run Locally

Clone the project

```bash
  git clone https://github.com/Samael-001/MyPortfolioReact
```

Go to the project directory

```bash
  cd MyPortfolioReact
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
